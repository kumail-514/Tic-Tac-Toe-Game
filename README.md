Description

This is a simple Tic Tac Toe game built using vanilla JavaScript, HTML, and CSS.
Two players (X and O) take turns to mark the board. The game continues until a player wins or the board is completely filled, resulting in a draw.

How to Play

Open index.html in your browser.

Click on the board to place your mark (X or O).

The game detects when a player completes a line of three marks (row, column, or diagonal) and declares the winner.

If the board is full and no one has won, the game declares a Draw.

Use the Reset button to start a new game anytime.

Controls

Click: Select a cell to place your mark

Reset Button: Restart the game at any time
